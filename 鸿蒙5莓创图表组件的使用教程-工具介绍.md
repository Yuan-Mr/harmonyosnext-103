### 基础使用步骤

莓创图表的宗旨就是引入方便，使用方便，更新方便。

1.  导入组件：

`import { McLineChart, Options } from '@mcui/mccharts'`

1.  初始化配置：

    @State options: Options = new Options({
      xAxis: {
        data: ['周一', '周二', '周三']  // 移除不必要的转义字符
      },
      series: [{
        name: '示例数据',
        data: [10, 20, 30]
      }]
    })

    build() {
      Row() {
        McLineChart({
          options: this.options  // 修正变量名一致性问题
        })
      }
      .height('50%')
    }

1.  动态更新：

局部更新数据：
`this.options.setVal({ series: \[{ data: \[50, 60, 70] }] })`
